# Calcolatore Voto Laurea - Ingegneria Aerospaziale 🚀

Un semplice script in Python per calcolare il voto di laurea finale e la fattibilità della lode per il corso di Laurea Magistrale in Ingegneria Aerospaziale, basato sul regolamento didattico ufficiale.

## Funzionalità
* Calcolo esatto del limite massimo di punti assegnabili in base alla media di partenza.
* Verifica automatica dei requisiti per la Lode (unanimità o maggioranza dei 2/3).
* Calcolo del voto di partenza in centodecimi ($M$).

## Utilizzo

Puoi avviare lo script in modalità interattiva:
```bash
python main.py
